Step 4: Requirements and Documents

Gather and prepare every core and program-specific document in the correct format before you begin uploading.


  • Core components: application form, certified transcripts, test score reports, 2–3 recommendation letters
  • Program-specific: CV/portfolio (arts/design), writing sample (PhD), APS certificate (Germany)
  • File preparation: convert to PDF, label clearly (e.g., Lastname_Firstname_Transcript.pdf), keep each ≤ 10 MB
  • Verify translation and certification requirements for non-English documents
