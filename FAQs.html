Student FAQs

What is the roadmap and how does it help me?
The roadmap is a personalized checklist of everything you need to apply—tests, transcripts, essays, recommenders, visa steps, and deadlines. It adapts to your chosen country, degree level, and target program dates, and sends reminders so you don’t miss key dates.
When should I start the application process?
Start early: Undergraduate (USA/Common App) — 12–18 months before enrollment; Master’s — 9–12 months; PhD — 12–24 months (contact supervisors early). Begin visa work 2–3 months before program start; some countries require earlier steps (e.g., blocked accounts, apostilles).
Which tests do I need and when should I take them?
Common tests: TOEFL/IELTS (language), SAT/ACT (US undergrad), GRE/GMAT (graduate). Check each program’s requirements, schedule tests at least 2–3 months before the earliest application deadline, and allow time for retakes.
How do I request transcripts and translations?
Request official transcripts from your university early (allow 4–8 weeks). If translations or apostilles are required, start those steps immediately—they can add several weeks. Ask your university for certified copies or electronic delivery when possible.
How many recommendation letters do I need and how do I manage them?
Most programs ask for 2–3 letters. Ask recommenders 6–8 weeks before deadlines, provide clear instructions and templates, and use the platform’s recommender upload link so they can submit directly. Track status (invited/submitted/overdue) through your dashboard.
How should I prepare my Statement of Purpose / personal statement?
Show fit (why this program), evidence of relevant experience, research interests (if applicable), and career goals. Tailor each statement to the program, respect word limits, get mentor feedback, and proofread multiple times.
What funding and scholarship steps should I take?
Search for university scholarships and external grants early—many scholarship deadlines are earlier than admissions deadlines. Prepare financial documents (bank statements, sponsor letters) and scholarship essays in parallel with applications.
What do I need for the visa application?
Typical visa requirements: acceptance letter, proof of funds, passport, health insurance, visa application form, and country-specific documents (e.g., blocked account for Germany). Start visa steps 2–3 months before program start, and consult the embassy website for exact checklists.
How should I format and name my documents?
Convert documents to PDF, name files like Lastname_Firstname_Type.pdf (e.g., Ivanov_Anna_Transcript.pdf), and keep each file under portal limits (commonly ≤10 MB). Keep versioned filenames for drafts (SOP_v1 → SOP_final).
What if I miss a deadline or my visa is delayed/denied?
If you miss a deadline: check rolling rounds, alternate intakes, or contact the program about extensions. If a visa is delayed/denied: review the reasons, contact the university about deferral, and consult a visa advisor. Your roadmap will surface contingency options when available.
Can recommenders upload letters for me?
Yes — the platform provides a secure upload link for recommenders so they can submit letters directly. You’ll receive notifications when letters are uploaded.
What should I do after I accept an offer?
Pay any required deposit, start the visa application, arrange housing and insurance, book flights aligned with orientation, and complete any pre-arrival enrolment steps. Register with local authorities and activate student services after arrival.
